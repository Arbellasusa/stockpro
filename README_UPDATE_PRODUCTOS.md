# StockPro — actualización de productos hoteleros

Incluye 84 productos hoteleros frecuentes agrupados por proveedor.

Archivos:
- `index.html`: app actualizada con catálogo precargado.
- `hotel_products_by_supplier.csv`: catálogo editable para importar/exportar.
- `hotel_products_by_supplier.json`: catálogo JSON para importación directa.
- `supplier_list.csv`: resumen de proveedores y categorías.

Uso:
1. Subir `index.html` al repositorio GitHub Pages.
2. Entrar a StockPro → pestaña DHS Supplier.
3. Presionar `Catálogo DHS`.
4. La app cargará solo SKUs nuevos y evitará duplicados.

Nota: los costos se dejaron en 0 para que el almacenero ponga los precios reales de factura. Los mínimos son valores sugeridos y editables.
